Regras para conhecimento sólido

  4. Anti tutorial-hopping — terminar um recurso antes do próximo.
  5. Tudo no Git — todo exercício commitado (treina Git + vira portfólio).
  6. Buffer — atrasou? empurra o cronograma, não corta a revisão.
